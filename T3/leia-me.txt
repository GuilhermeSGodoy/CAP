                                                                             INFORMAÇÕES GERAIS PARA USO
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
- O programa Gerador e Buscador de Caça-Palavras contém funções específicas para o sistema operacional Windows, portanto, não funciona adequadamente em outros sistemas operaci-
onais.
- A distribuição do programa é realizada através de uma pasta .zip, contendo o arquivo .c e o arquivo .txt. Estes arquivos não devem ser desmembrados.
- Existem arquivos de caça- palavras e lista de palavras com a temárica PALMEIRAS ofertados como exemplo na pasta .zip contendo o programa. 
- O programa Gerador e Buscador de Caça-Palavras apresenta sua melhor execução na IDE Dev-C++, versão 4.9.9.2, perfeitamente corrigida para problemas de compilação/execuçao. 
Saiba mais em (http://www.jpsiqueira.com.br/artigos/101-dev-c-como-instalar-e-resolver-problemas-com-windows-10).
- Para realizar a abertura de arquivos no programa é requerido:
	- Uso de arquivo na extensão .txt;
	- Existência do arquivo no mesmo diretório do programa;
	- Digitação no momento requisitado usando o seguinte formato: "nomedoarquivo.txt".
- O programa tolera a geração de caça-palavras usando números e caracteres, no entando, estes devem conter apenas um elemento por posição. (EX: é possível usar 1 ou c, mas não 
  12 ou ca).
- O programa Gerador e Buscador de Caça-Palavras é escrito na linguagem C e todos os testes devem estar adaptados para tal.

                                                                                  OBJETIVO PROGRAMA
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------   
- O objetivo direto do programa Gerador e Buscador de Caça-Palavras é satisfazer os critérios do Trabalho 3, elaborado pelo Prof. Dr. Ricardo Ciferri, em colaboração com o
Prof. Dr. Ednaldo Brigante Pizzolato.
Assim, o código pretende gerar três tipos de caça-palavras, para os quais são possíveis 4 tipos de geração de lista de palavras. Ainda, é necessário que o programa confira se
as palavras listadas estão ou não contidas na função.
Em níveis mais aprofundados, o programa pretende demonstrar o domínio adquirido pelos alunos para criação, manipulação e documentação de funções, procedimentos e outras habili-
dades trabalhadas na disciplina de Construçãode Algorítmos e Programação.
